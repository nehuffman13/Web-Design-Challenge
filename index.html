Welcome To Weather Analysis


The purpose of this project was to analyze how weather changes as you get closer to the equator. To complete the analysis, I pulled data from the OpenWeatherMap API to get a dataset on over 500 cities.

After I retrieved the dataset, I used Matplotlib to plot weather vs. latitude. Factors I looked at included:

Temperature

Cloudiness

Wind speed

Humidity


To see each visualization, you can choose each of them from the Plots dropdown button.
